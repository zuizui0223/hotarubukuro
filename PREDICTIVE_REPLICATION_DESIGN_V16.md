# 花色自然モデル予測複製 v16

## 目的

園芸起源を残差の大きさから推定するのではなく、測定済みの自然環境と空間構造だけから、現在と同じ観察設計で花色地図を反復生成する。その各地図へ観察地図と同じ候補抽出を適用し、観察された候補数、局所孤立性、人口環境、自然補正後の開花時期、有色花内強度が自然モデル下でどの程度再現されるかを調べる。

この解析が同定するのは「測定済み自然モデルでは再現しにくく、現地・履歴・遺伝解析を優先すべきセル」であり、園芸起源、逸出、遺伝子流動の確率ではない。

## 応答と解析粒度

- 候補単位は1-kmセルとする。
- 各セルの元画像数 `n_observations` を固定し、有色画像数 `n_pigmented` を二項応答として扱う。
- 混在セルを除外せず、0から `n_observations` までの有色数として使う。
- 有色花内強度は、有色が1点以上観察されたセルだけで別のGaussianモデルを推定する。
- 白花へ強度を割り当てず、白花のLab変動をアントシアニン量として解析しない。
- 開花日は花色候補の選択には用いず、環境、観察年、空間を用いた別のGaussianモデルから自然補正後の早咲き度を作る。

## 自然モデル

主解析は全国1,307セルを対象とする。

1. 発現モデル：`n_pigmented | n_observations ~ 50-km相対環境4軸 + SPDE`
2. 条件付き強度モデル：`conditional_intensity_median ~ 50-km相対環境4軸 + SPDE`
3. フェノロジーモデル：`median_DOY ~ 50-km相対環境4軸 + (観察年−2024) + (観察年−2024)² + SPDE`

観察年は2023–2025年に限られるため、未中心化の年と年²はほぼ完全相関になる。2024年で中心化してから二次項を作り、fold内で標準化する。

人口、道路、林縁、東西区分、DOY、有色内強度は発現候補モデルへ入れない。これらを候補選択に使ってから同じ特徴を証拠として数える循環を避けるためである。

## 空間交差検証と生成

- 既存の応答を見ずに作った100-km空間ブロック5-foldを固定する。
- 各foldの応答を除外してSPDEを推定し、held-outセルについて予測分布から1,000回生成する。
- メッシュは全観察座標から作るが、座標は応答ではないため応答漏洩とはしない。
- 切片と空間場平均を分離するため、各foldの学習地点に射影したSPDE場の平均を0に制約する。この制約は環境軸と空間場の完全な因果分離を意味しない。
- 固定効果、SPDE場、ハイパーパラメータを同じINLA posterior drawから取り、held-out fold内の空間相関を保つ。
- SPDEの観測セル予測は、メッシュ側の `Predictor` ではなく射影後の `APredictor = Aη` から抽出し、INLAの周辺事後平均との一致を全foldで監査する。
- fold間は異なる学習モデルから得た予測を同じ反復番号で連結したcross-fitted predictive mosaicであり、単一の全国同時事後分布ではない。局所孤立性は同じheld-out fold内の近隣だけで計算する。
- 観察地点とセルごとの観察数を固定する。未観察の全国ラスターセルへ候補なしを補完しない。

## 候補確率

観察された有色過剰について、セルごとに

`q+ = (1 + 予測複製で有色数が観察値以上だった回数) / (B + 1)`

を求める。白過剰は逆側の

`q- = (1 + 予測複製で有色数が観察値以下だった回数) / (B + 1)`

とする。候補順位は小さい `q` を優先し、同値の場合だけ予測標準偏差で標準化した方向付き超過量を用いる。人口、開花日、強度は順位を変えない。

絶対候補数には `q <= 0.05` を使う。上位5、10、20%は候補数を固定した感度系列であり、候補数の過剰検定には使わない。

## 各予測複製で繰り返す評価

各生成地図について、観察地図と同じ片側確率、適格条件、有色数による同値処理を適用する。比較する要約量は次に限定する。

- `q <= 0.05` の候補数
- 上位候補の25-km孤立割合
- 近隣5セルに対する方向付き花色孤立度
- 25-km人口量
- 自然補正後の早咲き度
- 有色候補だけの自然補正後強度
- 画像数と独立地点数。候補が疎な観察セルへ偏るかを検出する診断であり、園芸証拠ではない。

人口、自然補正後DOY、有色花内強度は候補選択から独立した観測済みファセット地図として固定する。各予測複製から得た候補集合が、その同じファセット地図のどこを選ぶかを帰無分布にする。別々に推定した発現・DOY・強度モデルの同じ反復番号を共同事後分布とはみなさない。DOY差と強度差は候補を作る残差応答ではなく、候補選択後の追跡特性に限る。

強度ファセットは、観測された有色花内強度があるセルだけを共通supportとし、観察地図と各予測複製の双方でそのsupport内に上位率を再適用する。白花セルへ強度を補完せず、欠測を無視した異なる母集団同士を比較しない。

観察値が1,000個の予測複製要約量のどこに位置するかを、95%区間、percentile、片側Monte Carlo確率とその標準誤差で示す。上位率とファセットを複数見るため、探索的な確率にはBH補正も併記する。

主要な多重比較familyは全国自然モデル内の方向・上位率・ファセットとする。共通supportの2モデルはBombus感度分析であり、主要familyへ重ねて数えない。全モデルをまとめたより保守的なBH値も補助的に残す。

「上位ほど特徴が強い」は目視で主張せず、上位5%と上位20%のファセット差を各予測複製でも同じように計算し、その入れ子差の帰無分布と比較する。

## Bombus感度分析

既存Bombus TIFFは使わず、再選択済み5種ENMeval予測だけを使う。5種すべてが揃う584セルに限定して、次を同じfoldと生成回数で比較する。

1. 共通support内の環境 + SPDE
2. 共通support内の環境 + SPDE + 5種群集fingerprint

群集fingerprintは、各学習fold内で環境4軸から直交化してから投入する。これは係数を各種の独立送粉効果として解釈するためではなく、共線性による数値不安定性を抑え、群集ブロック全体の追加予測情報を評価するためである。

ENMeval予測は相対的な生息適地支持と予測組成であり、個体数、訪花頻度、送粉効率、選択圧を表さない。全国主順位は5種共通supportへ制限しない。

## 解釈の上限

自然モデルからの逸脱は、少なくとも次の複数の説明を含む。

- 未測定の自然環境
- 局所的な遺伝構造、遺伝的浮動、分散史
- SNS撮影と登山道選択に由来する観察過程
- 発現、強度、開花日の条件付き独立仮定の不十分さ
- 園芸個体の植栽、逸出、交雑

したがって、人為的特徴が複数方向で自然複製範囲を外れても、「園芸影響と整合的な追跡候補」と記述する。園芸起源の実証には品種標本、栽培履歴、現地確認、集団遺伝解析が必要である。

## 再現性と停止条件

- 最終解析は各モデル1,000回、固定seedで行う。
- 前半500回と後半500回で候補順位Spearman相関と上位20% Jaccardを確認する。
- 主要入力の集約一致、全fold完走、予測値有限、候補確率範囲、共通support、選択変数漏洩を自動監査する。
- 結果と監査が確定するまで `final.Rmd` を変更しない。
